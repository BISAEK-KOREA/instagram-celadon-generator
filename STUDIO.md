# bisaek 콘텐츠 스튜디오 (지니 제작팀)

사우디 여성 대상 "한국문화×한류" 인스타 릴스를 자유 카테고리(문화·역사·음식·외국인의 한국생활·언어·여행 등)로 제작하는 팀 구조.

## 역할
| 역할 | 하는 일 |
|------|---------|
| 🧭 트렌드 스카우트 | 열린 카테고리에서 주제 발굴·후보 제안 |
| 📚 리서처(팩트체커) | 정확한 정보·출처·가격 수집, 사실 검증 |
| ✍️ 작가 | 3언어(아랍어/한국어/영어) 스토리·대본 작성 |
| 🎬 비주얼 디렉터 | 장면·무료 영상소재(Pexels) 선정, 화질 관리 |
| 🛠️ 프로듀서 | build_video.py로 영상 조립 + 제작번호 부여 + 갤러리 업로드 |
| ✅ QA·감수 | 사실·문화적 민감성(사우디 무슬림 정서)·저작권 점검 |
| 📤 퍼블리셔(지니) | 사장님 검수 통과분만 릴스로 게시 |

## 파이프라인
1. 주제 선정(자유 카테고리) → 제작번호 BSK-YYYYMMDD-NN 부여
2. 리서처+작가가 `projects/<slug>.json`(장면·3언어 대본·무료영상 키워드) 작성
3. `PROJECT_JSON=projects/<slug>.json python build_video.py` → 3언어 영상 생성
4. `docs/preview/`에 올리고 갤러리 페이지에 등록 → 사장님 검수
5. 승인분만 `post-reel.yml`로 게시

## 원칙
- 무료·저작권 안전 소재만(Pexels), 브랜드/인물 실사진은 사장님 제공 시에만
- 사실은 출처와 함께, 가격·수치는 "대략+시점" 명시
- 사우디 무슬림 여성 정서 존중(술·노출·민감 소재 배제), 긍정·정보형 톤
- 게시 결정은 항상 사장님(검수 후)
